Discussion 4:

Question:

As you start to build more complex web applications, you will likely come to find that you need to implement some type of software architecture pattern. One common example is when implementing user interfaces in our web apps, the Model View Controller (MVC) is commonly used.Perform some research about MVC and concisely explain how it is used. Here are some questions to consider addressing in your response: What are the benefits of using MVC? Are there any draw backs to using MVC? What are some of the popular web frameworks that implement a MVC approach? If you have formed one, what is your opinion about MVC? Be sure to cite your sources.
